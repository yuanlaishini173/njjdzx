技能人才评价证书全国联网查询
国家级 最权威

人社部官方指定的全国统一查询平台，可查询职业技能等级证书和技能人员职业资格证书。

📌 方式一：直接输入信息查询
  1. 打开网站：zscx.osta.org.cn
  2. 在首页选择你要查的证书类型（职业技能等级证书 / 技能人员职业资格证书）
  3. 按页面提示输入查询信息（姓名、证件号码、证书编号等）
  4. 输入验证码，点击「查询」按钮
  5. 即可看到证书详细信息
📌 方式二：登录后查看所有证书
  1. 点击页面右上角「登录」
  2. 选择登录方式：账号密码 / 电子社保卡扫码 / 掌上12333扫码
  3. 登录后进入个人中心
  4. 找到「我的证书」或「证书查询」栏目
  5. 可查看你名下所有的证书列表
  • 覆盖范围：全国所有经备案机构颁发的证书
  • 数据来源：人社部官方数据库

❓ 常见问题

查询证书时可能遇到的问题，都在这里了

电子证书和纸质证书一样有效吗？ +

一样有效！

电子证书和纸质证书具有同等法律效力，可以用于求职、晋升、申领补贴、落户等所有场景。

现在很多地方已经全面推行电子证书，不再发放纸质证书了。用江苏智慧人社 APP 或江苏人社网上办事大厅就能下载电子证书。

证书能领补贴吗？怎么领？ +

可以领！取得职业技能等级证书后，符合条件的可以申领技能提升补贴。

申领条件（参考）：

• 在企业累计缴纳社保满 12 个月
• 或者正在领取失业保险金
• 证书核发之日起 12 个月内申报

申领渠道：

• 电脑端：江苏人社网上办事服务大厅 → 技能提升补贴申领
• 手机端：江苏智慧人社 APP → 技能提升补贴申领

补贴金额根据等级不同，一般初级 1000 元、中级 1500 元、高级 2000 元左右，具体以当地政策为准。

怎么判断证书是不是真的？ +

最可靠的方法就是在官方渠道查询：

1. 优先查「技能人才评价证书全国联网查询系统」（zscx.osta.org.cn），能查到就是真的
2. 也可以扫证书上的二维码验证
3. 查不到的可以联系发证机构核实

⚠️ 重要提醒：凡是说"免考包过""快速取证"的基本都是骗局，正规证书必须本人参加考试！所有官方查询渠道都是免费的！

我在外地考的证书，江苏能查到吗？ +

可以的！

职业技能等级证书和职业资格证书都是全国通用的。只要是正规备案的评价机构发的证，在「技能人才评价证书全国联网查询系统」上都能查到，不受地域限制。

跨省也能领取补贴吗？具体要看参保地的政策，一般在社保缴纳地申领。

查不到证书信息怎么办？ +

先别着急，按以下步骤排查：

1. 等一等：考完试后，证书数据上传需要时间，一般 30-45 个工作日
2. 核对信息：检查姓名、身份证号有没有输错，身份证末位 X 要大写
3. 多试几个平台：先查省级平台（江苏人社网），再查国家级平台
4. 联系发证机构：确认数据是否已上报
5. 拨打 12333：工作日工作时间拨打全国人社服务热线 12333 咨询
