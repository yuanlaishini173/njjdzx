先搞懂：三个核心概念

💬

指令 / Prompt

你对 AI 说的话、下达的命令。指令是最基础的交互方式， 每个 AI 工具都支持，但格式和特殊指令各有不同。

"帮我写一个登录页面的 HTML 代码"
⚡

技能 / Skill

预设好的、可以复用的"专业能力包"。相当于给 AI 请了个专家， 一键就能获得某个领域的专业能力，不用每次都从头教。

Trae Skill: 生成可编辑 PPT
🔌

插件 / Plugin

让 AI 能"动手做事"的扩展工具。插件可以让 AI 连接外部服务、 读取实时数据、操作软件，突破纯对话的限制。

联网搜索、读取文件、调用 API

💡 简单理解：指令是你说的话， 技能是 AI 的专业能力， 插件是 AI 能用的工具

深入对比：指令系统

每个 AI 工具都可以用自然语言对话，但各自有独特的指令格式和特殊命令

🤖
ChatGPT
OpenAI 出品

💬 指令特点

  • 支持 @ 提及 GPTs 调用特定技能
  • Custom Instructions 自定义系统提示
  • 支持多模态（图片、语音输入）
  • 指令跟随能力强，擅长结构化输出

📌 特殊指令

  • 系统级指令：/imagine（DALL·E 绘图）
  • 支持 Markdown 格式指令
  • 可通过 System Prompt 定制 AI 人设
✅ 优点
  • 生态最成熟
  • 指令遵循度高
  • 多模态支持好
❌ 缺点
  • 国内无法直接访问
  • GPTs 需 Plus 会员
  • 中文能力一般
🧠
Claude
Anthropic 出品

💬 指令特点

  • 长文本理解能力极强（200K+ 上下文）
  • 擅长深度分析和逻辑推理
  • Artifacts 可生成可交互内容
  • 支持多文件、多格式输入

📌 特殊指令

  • / 开头的斜杠命令
  • Claude Code 专用开发指令
  • 项目级规则配置
✅ 优点
  • 长文本处理强

❌ 缺点

Trae
字节跳动出品

❌ 缺点

深入对比：技能生态

技能（Skills / GPTs / 智能体）让 AI 获得专业领域能力，不用每次从零开始

🤖

ChatGPT → GPTs

GPT Store 是最早的 AI 技能商店，用户可以创建和分享定制化的 GPT。 涵盖学习、编程、创意、工作等各领域。

需 Plus 会员才能创建/使用
🌟

Trae → Skills

Trae 的 Skills 是可复用的专业能力包，能完成特定复杂任务。 内置技能商店，也可以自己创建 Skill。

PPT生成、文档生成、数据分析等
✨

豆包 → 智能体 (Bot)

豆包的智能体生态丰富，支持自定义人设、知识库、插件。 社区有大量用户分享的 Bot 可以直接使用。

免费创建，社区共享
🌊

文心 → 智能体 / 插件

文心一言的智能体（Agent）支持工作流编排，可以串联多个步骤。 插件和智能体结合，能力更全面。

Agent Builder 可视化创建
🧠

Claude → Projects

Claude 的 Projects 功能更偏向项目级定制， 可以上传文档和知识，让 AI 围绕特定项目工作。

项目级知识库 + 定制指令
💻

Cursor → Rules

Cursor 的 Rules 是项目级配置文件， 可以定义代码风格、架构规范等，让 AI 遵循你的规则。

.cursorrules 文件配置

🔮 MCP：正在统一的插件标准

MCP（Model Context Protocol）是由 Anthropic 提出的开放协议，正在成为 AI 工具连接外部服务的统一标准。 Trae 等新一代 AI IDE 已经率先支持 MCP，未来可能会出现"一次开发，多平台使用"的插件生态。 就像 USB 统一了各种接口一样，MCP 有望统一 AI 工具的插件体系。

常见问题

新手应该从哪个 AI 工具开始？ +

如果你在国内，推荐从 豆包 或 Trae 开始：

• 纯对话、写文案、头脑风暴 → 豆包（简单易用，免费额度高）
• 想写代码、做开发 → Trae（IDE 集成，SOLO 模式强大）

如果你能翻墙，可以试试 ChatGPT 和 Claude，它们的能力更全面，但国内访问不方便。

指令、技能、插件，我该先学哪个？ +

建议学习顺序：指令 → 技能 → 插件

• 指令：最基础，每天都要用，花 1-2 天了解基本技巧即可
• 技能：遇到特定领域问题时再找对应的技能，不用提前学
• 插件：当你发现 AI"光说不练"时，再去研究插件扩展能力

核心原则：按需学习，用的时候再深入，效率最高。

国内工具和国际工具差距大吗？ +

要看具体场景：

• 中文能力：国内工具（豆包、文心、Trae）普遍更好
• 代码能力：国际工具（GPT-4o、Claude 3.5）略强，但差距在缩小
• 生态丰富度：ChatGPT 的 GPTs 和插件生态最成熟
• 访问便利性：国内工具完胜，速度快、无需翻墙

对于大多数国内用户，国产工具完全够用，而且在快速进步中。

可以同时用多个 AI 工具吗？ +

当然可以！而且推荐这么做。

不同工具有不同优势，组合使用效果最好：

• 日常对话、写文案 → 豆包
• 写代码、做项目 → Trae / Cursor
• 深度分析长文档 → Claude
• 复杂问题拿不准 → 多问几个工具，对比答案

就像你不会只用一种 App 一样，AI 工具也可以组合使用。

什么是 SOLO 模式？和普通 AI 有什么区别？ +

SOLO 模式是 Trae 的特色功能，简单说就是：

普通 AI：你说一步，它做一步，需要你全程指挥
SOLO 模式：你说目标，AI 自己规划、执行、测试，全流程自动化

就像从"手动挡"变成"自动驾驶"，你只需要说清楚目的地， AI 会自己规划路线、开车、遇到问题自己调整，最后把成果交付给你。 特别适合从零开始做一个完整项目的场景。

我需要付费买会员吗？ +

看你的使用频率和需求：

• 偶尔用用：免费版完全够用，豆包、Trae 免费额度都很高
• 每天都用：建议付费，体验会好很多（速度更快、功能更多）
• 做开发：Trae 或 Cursor 的付费版值得投资，效率提升明显

建议先用免费版体验，觉得真的能帮到你再付费，不要冲动消费。
